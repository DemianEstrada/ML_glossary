# ML_glossary

Personal reference repository containing Machine Learning models organized by folder.  
Each model includes:

- `math.md`: mathematical explanation, assumptions, hyperparameters, and diagnostics  
- `example.ipynb`: complete end-to-end example  
- `template.py`: reusable template for future projects  

## Structure

- `models/`: models organized by folder  
- `src/`: reusable utilities  
- `docs/`: guides and conventions  

## How to run

```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
